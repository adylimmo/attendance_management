# student attendance_management web application

## Features

- Supper Admin
  - Controls everything
- Teachers Profile
  - Update Profile
  - Forgot Password
- Teachers Login
- Class Add
- Student Add by class
- Student remove
- Generate attendance report
- Download report in excel format

# Technology

- Frontend

  - HTML
  - CSS
  - JAVASCRIPT
  - BOOTSTRAP (If needed)

- Backend
  - GOLANG
    - gorilla/mux (for router)
    - gorilla/session (for login authentication)
- Database
  - POSTGRESQL

- ORM
    - gorm (http://gorm.io)
